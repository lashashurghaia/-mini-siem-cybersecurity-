# საბაკალავრო პროექტი: ქსელური Mini-SIEM სისტემა მანქანური სწავლების გამოყენებით

ეს პროექტი შევქმენი კომპიუტერული მეცნიერების საბაკალავრო პროგრამის ფარგლებში. ჩემი მიზანი იყო შემექმნა მარტივი, მაგრამ რეალურად მომუშავე SIEM (Security Information and Event Management) სისტემის პროტოტიპი, რომელიც ხელოვნური ინტელექტის (Machine Learning-ის) დახმარებით შეძლებდა ქსელში საეჭვო აქტივობებისა და თავდასხმების აღმოჩენას, რასაც მოჰყვებოდა ავტომატური რეაგირება.


## როგორ არის აგებული სისტემა

პროექტი დავყავი სამ ძირითად ნაწილად, რათა კოდი ყოფილიყო მოწესრიგებული და მარტივად წასაკითხი:

1. მონაცემების მომზადება (generate_data.py): რადგან რეალური ქსელური ტრაფიკის ბაზები ხშირად ძალიან დიდია ან კონფიდენციალურია, დავწერე სკრიპტი, რომელიც თავად აგენერირებს საჭირო მონაცემებს. ის ქმნის როგორც ჩვეულებრივი, უსაფრთხო მომხმარებლების ქცევას, ასევე შეტევების იმიტაციას (მაგალითად, DDoS, Brute-Force ან Port Scan სცენარებს) და ქმნის 5%-იან შემთხვევით ხმაურს რეალისტურობისთვის.

2. მოდელის გაწვრთნა (train.py): აქ გამოვიყენე Scikit-learn ბიბლიოთეკა და Random Forest-ის ალგორითმი. კოდი კითხულობს გენერირებულ მონაცემებს, სწავლობს მათ მახასიათებლებს და მიღებულ მოდელს ინახავს cyber_model.pkl ფაილში, ხოლო სიზუსტის მაჩვენებლებს აგროვებს metrics.json-ში.

3. ვებ პანელი (app.py): მომხმარებლის ინტერფეისი ავაწყვე Streamlit-ზე. 

## მართვის პანელის ფუნქციები

საიტზე გადასვლისას მარცხენა მენიუდან შესაძლებელია სამი სხვადასხვა გვერდის გამოყენება:

1. რეალური დროის მონიტორინგი: აქ ხდება ქსელური ნაკადის იმიტაცია. ღილაკზე დაჭერით სისტემა იწყებს 20 პაკეტის გატარებას, რეალურ დროში ამოწმებს მათ მოდელის დახმარებით, გამოიტანს შესაბამის ბლოკებს და ბლოკავს მავნე IP-ებს.

2. პრევენციის ისტორია: ეს არის ჟურნალი, სადაც SQLite მონაცემთა ბაზიდან ავტომატურად იწერება და იკითხება ყველა ის ინციდენტი, რომელიც სისტემამ მუშაობის პროცესში დააფიქსირა ან დაბლოკა. აქვე დამატებულია ლოგების CSV/JSON ფორმატში გადმოწერისა და ბაზის გასუფთავების ფუნქციები.

3. მოდელის ანალიტიკა: აქ გამოვაჩინე მოდელის მუშაობის საერთო სიზუსტე (Accuracy) და შეცდომების მატრიცა (Confusion Matrix) როგორც გასაგები ცხრილის, ისე ნახაზის სახით, რაც საჭიროა იმის სანახავად, თუ რამდენად სწორად მუშაობს ალგორითმი.

## რა არის საჭირო პროექტის გასაშვებად

სისტემის ლოკალურად ჩასართავად საჭიროა რამდენიმე ბიბლიოთეკის დაყენება და ფაილების რიგითობის დაცვა.

გახსენით ტერმინალი იმ საქაღალდეში, სადაც ფაილებია და მიყევით ამ ნაბიჯებს:

1. ბიბლიოთეკების ინსტალაცია: pip install streamlit pandas numpy scikit-learn joblib matplotlib
2. ბაზის შექმნა: python generate_data.py
3. მოდელის გაწვრთნა: python train.py
4. საიტის ჩართვა Streamlit-ით: streamlit run app.py
